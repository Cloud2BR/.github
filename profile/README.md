> Cloud2BR (Cloud to be Ready) is a `consulting and enablement hub` dedicated to helping organizations explore, validate, and adopt Microsoft cloud solutions. Specialize in PoC (proof-of-concept) projects, sandbox environments, training sessions, and technical events that empower teams to learn, experiment, and innovate.

## What You'll Find Here

- Starter kits and templates: reusable code and reference architectures for proof-of-concepts
- Sandbox projects: experimental environments for testing and learning
- Tech talks and webinars: supporting materials, demos, and examples
- Guides and documentation: practical resources for workshops and events
- Utilities and tools: scripts and helpers to accelerate cloud adoption

## How We Work

- Experiment: build and validate ideas quickly in sandbox environments
- Enable: provide training, talks, and resources to empower teams
- Scale: connect clients with trusted Microsoft partners to move from MVP to production

## Get Involved

- Explore repositories for PoC templates and learning resources
- Use guides to run your own sandbox experiments
- Join events and webinars to deepen cloud knowledge

> [!NOTE]
> - Content in Cloud2BR repositories is intended for learning, demonstrations, and proof-of-concepts. Review and validate before using in it.
